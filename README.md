# QueueBit

## Steps to run the project

1. Clone the project
   ```sh
   git clone <repo-url>
   cd QueueBit
````

2. Install root dependencies

   ```sh
   npm install
   ```

3. Install client dependencies

   ```sh
   cd client
   npm install
   cd ..
   ```

4. Install owner dependencies

   ```sh
   cd owner
   npm install
   cd ..
   ```

5. Run the project

   ```sh
   npm run dev
   ```
